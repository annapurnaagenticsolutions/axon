# AXON Task #03 — FastMCP Code Generator
> Depends on Tasks #01 and #02. Hand this ticket to any capable coding model.

---

## BACKGROUND

AXON is an AI-native programming language. We have a parser (Tasks #01, #02) that
converts `.ax` source files into Python dataclass AST nodes. This task builds the
**code generator** that takes those AST nodes and emits a working FastMCP Python server.

The goal: `axon serve myagent.ax` → launches an MCP server that Claude Desktop can connect to.

---

## WHAT TO BUILD

One Python file:

`src/axon/codegen/mcp.py` — generates FastMCP server code from parsed AXON declarations.

## RULES & CONSTRAINTS

1. **Only emit tools that appear in the agent's `tools:` list.** If a `ToolDecl` exists but is not referenced by any agent, still emit it (it may be useful as a standalone tool).
2. **Function names must be snake_case.** `FetchIssues` → `fetch_issues`.
3. **Docstrings:** combine all `///` docstring lines with newlines. Append a blank line, then `AXON source: Name(params) -> ReturnType`.
4. **Body:** always emit as a comment prefixed with `# AXON body:`. Never try to translate the body.
5. **Raise NotImplementedError** in every tool body stub.
6. **Type mapping:** use `type_to_python()`. Params become function arguments. Return type becomes the function return annotation.
7. **Literal union types** (`"a" | "b"`) map to `str` in Python.
8. **`Result<T, E>` always maps to `dict`** for simplicity in Phase 1.
9. **`()` maps to `None`** (unit return type).
10. **Agent metadata:** emit as module-level constants (`AXON_AGENT_TOOLS`, `AXON_AGENT_MEMORY`, etc.) so tooling can inspect without running the server.
11. **Header comment:** always include the `Generated by AXON v0.1` docstring at the top.
12. **If there are multiple agents** in one file, generate one MCP server using the name of the first agent found. Emit a warning comment if multiple agents are present.
13. **No runtime imports beyond `fastmcp`, `typing`, `os`.** Do not import anthropic, openai, or httpx in generated code.

from axon.parser import parse
from axon.codegen.mcp import generate_mcp_server, type_to_python, to_snake_case

def test_snake_case():
    assert to_snake_case("FetchIssues") == "fetch_issues"
    assert to_snake_case("WebSearch") == "web_search"
    assert to_snake_case("AddLabel") == "add_label"
    assert to_snake_case("readCSV") == "read_csv"

def test_type_to_python_primitives():
    assert type_to_python("Str") == "str"
    assert type_to_python("Int") == "int"
    assert type_to_python("Float") == "float"
    assert type_to_python("Bool") == "bool"
    assert type_to_python("Any") == "Any"
    assert type_to_python("()") == "None"

    assert type_to_python("List<Str>") == "list[str]"
    assert type_to_python("Option<Int>") == "Optional[int]"
    assert type_to_python("Map<Str, Int>") == "dict[str, int]"
